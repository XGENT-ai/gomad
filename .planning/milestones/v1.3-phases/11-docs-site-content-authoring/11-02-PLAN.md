<objective>
Ship the auto-gen injector (`tools/inject-reference-tables.cjs`), its unit tests (`test/test-inject-reference-tables.js`), and the pipeline patch (`tools/build-docs.mjs`) that runs it before link-check. The injector reads `src/gomad-skills/*/SKILL.md` (and `src/core-skills/*/SKILL.md`), renders markdown tables for the agents and skills reference pages, and substitutes the rendered content between `<!-- AUTO:agents-table-{start,end} -->` and `<!-- AUTO:skills-table-*-{start,end} -->` markers.

Purpose: Eliminate drift on the moving-target persona/skill catalog (D-05). Authors keep markers in source; the injector populates tables on every build.
