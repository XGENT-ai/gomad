<objective>
Fix two install-time bugs in `@xgent-ai/gomad` v1.3.1:

  1. **bug1** — Claude Code installs erroneously create `.claude/skills/gm:agent-*` directories that duplicate the launcher commands at `.claude/commands/gm/agent-*.md`. Per project rules, on Claude Code agent personas live ONLY as commands (via `writeAgentLaunchers`), not as skills. Codex (and other non-launcher platforms) keep installing them as skills under `.agents/skills/` — that path is unchanged.

  2. **bug2** — Upgrading from a v1.2-or-earlier install (where the agile module was coded `gomad`) leaves an orphaned `_gomad/gomad/config.yaml` and empty `_gomad/gomad/` parent dir. The legacy-id migration in `existing-install.js:98` rewrites `gomad → agile` on read, hiding the orphan from `_removeDeselectedModules`. The v1.2 branch in `cleanup-planner.js` only handles `_gomad/gomad/agents/*.md`, missing `config.yaml` and the parent dir.

Purpose: Eliminate two cosmetic+correctness regressions from the v1.3 line. bug1 is high-severity for fresh Claude Code installs (8 unwanted dirs + agent listing collisions). bug2 leaves dirty state in every upgraded install.

Output: Three surgical edits + two test suites + npm script wiring. No public API changes; no manifest schema changes; no new D-XX decision records (these are bug fixes against existing invariants D-29, D-31, D-34, D-42).

    // installs survives because existing-install.js:98 maps `gomad → agile` on
    // read, hiding the dir from _removeDeselectedModules. Snapshot+remove it

    let v12ChildrenCleared = plan.to_remove.length > 0; // true if persona loop above queued anything

    const legacyModuleConfig = path.join(workspaceRoot, '_gomad', 'gomad', 'config.yaml');
    if (await fs.pathExists(legacyModuleConfig)) {
      let resolved;
      try {
        resolved = await fs.realpath(legacyModuleConfig);
      } catch (error) {
        if (error.code !== 'ENOENT') throw error;
        resolved = null; // race: removed mid-flight
      }
      if (resolved) {
        if (!isContained(resolved, workspaceRoot)) {
          await prompts.log.warn('SYMLINK_ESCAPE: ' + legacyModuleConfig + ' → ' + resolved + ', refusing to touch');
          plan.refused.push({ idx: null, entry: { path: legacyModuleConfig }, reason: 'SYMLINK_ESCAPE' });

          const orig_hash = (priorEntry && priorEntry.hash) || null;
          let was_modified = null;
          if (orig_hash && hashGen) {

            was_modified = currentHash !== orig_hash;

            install_root: '_gomad',
            relative_path: 'gomad/config.yaml',
            orig_hash,
            was_modified,

          plan.to_remove.push(resolved);
          handledPaths.add(resolved);
          v12ChildrenCleared = true;

    // Empty parent dir removal — only if at least one child was queued.
    // No snapshot (empty dirs carry no recoverable content).
    if (v12ChildrenCleared) {
      const legacyParent = path.join(workspaceRoot, '_gomad', 'gomad');
