The script has two phases:

**Phase 1: Forbidden path check (D-05)**
- Run `npm pack --dry-run 2>&1` and capture output
- Parse `npm notice` lines to extract file paths (strip the `npm notice  <size>  ` prefix)
- Check each path against forbidden patterns: `/^\.planning\//`, `/^test\//`, `/^\.github\//`, `/^docs\//`, `/^website\//`, `/banner-bmad/`
- If any match, print `FAIL: forbidden file in tarball: <path>` and set failure flag

**Phase 2: Grep-clean check (VFY-03)**
- Run `grep -rliE "\\b(bmad|bmm)\\b" src/ tools/installer/ --include="*.js" --include="*.yaml" --include="*.md" --include="*.json" --include="*.csv" 2>/dev/null`
- Filter out `LICENSE` and `CHANGELOG.md` from results
- If any files remain, print `FAIL: residual bmad/bmm references found in: <files>` and set failure flag

**Exit:**
- If any failures: `process.exit(1)`
- If all pass: print `OK: <N> files in tarball, no forbidden paths, no bmad/bmm residuals` and exit 0

Use `execSync` from `node:child_process` for both npm pack and grep commands. Handle grep returning exit code 1 (no matches) as success via try/catch.
